# Grabber 

> Grabber是基于Java和C++开发的一门新型编程语言。

----

## 如何构建出Grabber的编译器？

开发环境以及编译环境要求

- JDK1.8或JDK1.8以上版本

- GCC97或以上GCC编译器环境

- NASM汇编编译器2.11.08及以上

设置环境变量，以及配置
```
  GRABBER_JVM_HOME = "YOUR JDK PATH"
  GRABBER_NASM_HOME = "YOUR NASM PATH"
  GRABBER_GCC_HOME = "YOUR GCC PATH"
```
在Grabber的项目根路径下的`config`文件夹中可以看到对于Grabber编译器的一些配置。
